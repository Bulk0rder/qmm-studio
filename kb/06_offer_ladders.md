# Offer Ladders

**Definition**: Structuring value so customers can ascend from "Free" to "VIP".

## The Rungs
1. **Lead Magnet (Free)**: Cheatsheet, Webinar, Audit. Low friction.
2. **Tripwire (Low Cost)**: N1000 - N5000. Changes relationship from "Lurker" to "Buyer".
3. **Core Offer**: The main product.
4. **Profit Maximizer**: Upsells, Cross-sells, Premium plans.

## B2B Example
- **Magnet**: "2025 Industry Report PDF".
- **Tripwire**: "Paid Strategy Session (Discounted)".
- **Core**: "Retainer Contract".
